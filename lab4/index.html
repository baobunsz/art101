Results

Tasks to create a character's 3D model:

  • Watch instructor’s tutorial
  • Sketch a blueprint
    • Create the front and side view of the character
  • Open Blender
  • Import images
    • Align the bottom of character to the axis
  • Lock images
  • Add cylinder shape
    • Adjust amount of faces on the cylinder
  • Build up your character
    • Turn on mirror add-on to build up the second half of your character
  • Connect vertices when they meet up to create one solid figure

Tasks for the game "Snake":

  1. Print the map
  2. Accept user’s input
  3. Move the snack
  4. Analyze actions
    • If impact with apple, make snake longer
    • If impact with walls or itself, game over
